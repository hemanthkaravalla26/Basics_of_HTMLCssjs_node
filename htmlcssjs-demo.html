Progressive HTML + CSS + JS Demo

This page is structured for teaching: each section shows an example and explains it briefly.

Part 1 — HTML Basics

1. Headings

Main Heading

Sub Heading

Explanation: Headings (<h1> to <h6>) are used for titles and sub-titles. H1 is largest, H6 is smallest.

2. Paragraph

This is a paragraph element. It contains blocks of text for reading.

Explanation: Paragraphs (<p>) are the basic block of text in HTML.

3. Link

Visit Google

Explanation: The <a> tag creates hyperlinks. target="_blank" opens them in a new tab.

4. Image

Explanation: The <img> tag displays images. The alt text shows if the image fails to load.

5. Lists

  • Apple
  • Banana
  1. First
  2. Second

Explanation: Unordered lists (<ul>) use bullets; ordered lists (<ol>) use numbers.

6. Table

NameAge
John25
Jane30

Explanation: Tables (<table>) arrange data in rows (<tr>) and columns (<td>). Headers use <th>.

7. Form

Name:

Email:

Explanation: Forms collect user input. The input tag changes behavior based on type.

8. Audio

Explanation: The <audio> tag embeds audio. The controls attribute adds play/pause buttons.

9. Video

Explanation: The <video> tag plays videos. You can set width, height, and controls.

10. Div & Span

This is a styled box (div)

This is red text inside a paragraph.

Explanation: <div> is a block container; <span> is inline, for styling parts of text.

Part 2 — CSS Basics

1. Colors & Fonts

This text is green with Verdana font.

Explanation: CSS changes appearance. color sets text color; font-family changes font style.

2. Background & Borders

This box has background color and a border.

Explanation: background-color fills the area; border outlines it.

3. Margin & Padding

This box has margin (outside space) and padding (inside space).

Explanation: margin adds space outside; padding adds space inside.

4. Hover Effect

Hover over this link

Explanation: :hover applies styles when the mouse is over the element.

5. Styled Buttons

Explanation: Buttons can be styled with colors, padding, and hover effects.

Explanation: JS can respond to events like clicks. alert() shows a popup.
